1.x -- 第 1 章总结

章节复习

语句（statement）是让程序执行一些动作的一种指令。语句一般以分号结尾。

函数（function）是按顺序执行的一组语句。每个 C++ 程序必须拥有一个叫 main 的特殊函数。运行程序时，从 main 函数的顶部开始执行。

在编程上，函数（对象，类型，模板等）的名字叫做 标识符（identifier）。

决定 C++ 语言中的元素如何组织的规则叫做 语法（syntax）。语法错误（syntax error）是违反语言的语法规则时的错误。

程序员使用 注释（comment） 在代码中做笔记。行注释以 // 开始直到行尾。块注释以 /* 开始，到匹配的 */ 结束。块注释不能嵌套。

可以使用注释来暂时禁用部分代码。这叫做把代码注释掉。

数据（data）是可被计算机移动、处理、存储的任何信息。单独的一个数据叫做 值（value）。值的常见例子有字母（a）、数字（5）、文本（Hello）。

变量（varible）是一块具名的内存，可以用来存储值。创建变量的语句叫 定义语句（definition statement）。程序运行时，每个变量会被 实例化（instatiated），意思是给它分配了一个内存地址。

数据类型（data type）告诉编译器如何把一段数据解释成有意义的值。整数（integer）是没有小数部分的数。

通过 operator= 的 复制赋值（copy assignment）可用来给已存在的变量赋值。

初始化（initialization）可在创建变量时给它一个值。C++ 支持三种初始化：

  • 复制初始化
  • 直接初始化（圆括号初始化）
  • 列表初始化（统一初始化，花括号初始化）

应当倾向于使用列表初始化。应当使用初始化而不是事后赋值。

尽管可以在单独一条语句里定义多个变量，但每个变量都在单独的一行用单独的语句定义会更好。

std::cout 与 operator<< 可用于把表达式以文本形式输出到控制台。std::endl 输出换行符，让控制台光标移到下一行。std::cin 与 operator>> 可用于从键盘读取输入。

还没有得到值的变量叫做 未初始化的变量（uninitialized variable）试图获取未初始化的变量值会造成 未定义行为（undefined behavior）。还有很多种方式造成未定义行为。

C++ 保留了一些名字，称为 关键字（keywords）。它们在语言里有特殊含义，不能用作变量名。

字面值常量（literal constant）是直接插入源代码的固定值。例如 5 和 Hello world!。

运算（operation） 是包含零个或多个称为 操作数（operands）的输入值的过程。运算的结果是产生一个输出值。

一元（unary）、二元（binary）、三元（ternary）、零元（nullary）运算符分别接受一个、两个、三个、零个操作数。其中二元运算符的操作数又称为左操作数和右操作数。

表达式（expression）是字面值、变量、运算符、函数调用的组合，求值会得到一个单独的输出值。计算这个值的过程叫做 求值（evaluation），产生的值称为表达式的 结果（result）。

表达式语句（expression statement）是表达式后加分号得到的语句。

写程序时，写几行或者一个函数就编译、解决错误、保证运行正常。不要等到全写完了再第一次编译。

专注于让代码能正确运行。一旦能确信某段代码一定会保留之后，就可以花点时间移除（或注释掉）临时代码和调试用代码、添加注释、处理错误情况、格式化代码、确保遵循了最佳实践、移除多余的逻辑等等。

第一遍写出来的程序经常是混乱且不完美的。绝大多数代码需要整理和改进来成就卓越。
