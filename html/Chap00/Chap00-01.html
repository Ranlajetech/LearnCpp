0.1 -- 教程简介

欢迎！

欢迎阅读本 C++ 教程！本教程的最高目标就是让学习 C++ 变得简单。

本教程不假设你有编程经验，你需要知道的这里都会教，并附上大量示例。

无论你是作为爱好还是作为事业来学习 C++，你都来对地方了！

课程结构

这一绪论章旨在让你对 C++ 是什么、怎么来的、程序如何运行有个大概了解，并带你安装编程所需的软件。你将编写自己的第一个程序。

后面的章节将探索 C++ 语言的各个方面。在第 1 章中，你将粗略地了解 C++ 的一些基础概念，然后就可以写点很简单的程序了。之后会深入讲解它们，还会引入好多新概念。

每章都有个主题，里面每一节都和这个主题相关。我们不规定你要在一节课上花多长时间，按自己的节奏来就好。

课程目标

开始学习之前，先介绍一下本教程的几个重要目标。

  • 在介绍 C++ 的同时也会说一说编程相关的一些附带问题。传统的编程教材在教编程语言基础这一点上都做得很好，但它们经常会忽略实际编程中的附带问题。比如说，很多教材都没有编程风格、常见隐患、调试、好 / 坏编程实践、程序测试这样的章节。因此当你学完了那本书，你知道该怎么编程，但你可能会无意间养成一些让你以后很难受的坏习惯。本教程的目标之一就是穿插着讲解这些附带内容，你学完之后不仅要知道编程的时候应该怎么做，也要知道有哪些事情不应该做，两者同样重要。

  • 提供大量示例。很多人都会从示例中学到和从教程文本中一样多甚至更多的知识。本教程会努力提供大量简明的例子来展示刚学的编程概念该怎么用。我们也会尽可能在例子中避免两个烦人的东西：省略号(...)和不作解释的新概念。前者为了节省空间而省略了程序的必要部分，而后者是例子中必要的一部分但没作任何解释。它们都有可能让人搞不清然后卡住。

  • 提供课后练习题。很多章节都会包含一些要求你完成的练习题，同时附有答案。你可以做完对下答案，看看你的解答和我们提供的答案有何异同，也可以在卡住的时候看看答案是怎么做的，然后自己回去重新做一遍，多在卡住的地方下点功夫。（译者注：练习题部分没做搬运，请移步原文网站的对应章节查看）

  • 最重要的是要开心。编程是让人愉悦的，如果你编程的时候感觉并不开心，那么你此时的心态不适合编程。疲惫、不开心的程序员更容易犯错，而后期调试 debug 比一开始就写对要多花好几倍的时间。这时候你该去歇一歇，好好睡一晚上，然后精神抖擞地继续编程。

如何高效利用本教程

阅读本教程时，下面这些建议能帮你更高效地学习。

  • 把例子纯手打输入到编辑器里然后自己编译。别复制粘贴！做到这一点，你可能会发现某些你经常犯错的地方，也会更熟悉编译器发出的警告和错误。同时也不要无脑照抄，要一边打字一边想这行代码是干什么的，它对于整个程序有什么作用。如果想不通，那这就是需要你深入研究的地方。

  • 当你犯了错或者找到了你程序中的 bug 时，尝试自己修复它。你应当在寻求帮助之前先尝试自己解决。学会怎么找到并修复错误是编程的重要一环。不要抵触学习怎么使用调试器（以后会详细解释的），它是帮你定位程序错误的有力工具。

  • 用例子做做实验。你可以改改程序里的数字看看会发生什么，也可以改改程序功能让它做更多的事情（比如一个程序是把两个数加一起，你可以把它改成三个数相加）。你甚至可以想办法看看怎么能让程序崩溃（程序让你输入一个数，你可以试试输入字符）。在改程序的过程中你会学到更多的东西。

  • 花点时间做做习题。如果你是编程新手，你会发现这些习题还是挺有挑战性的（很正常，因为你的大脑还在适应编程这件事）。不要因为你第一次做出来的答案不对就气馁。你可能要在成功到达终点之前尝试好几条路径。如果你真的卡住做不出来了，没关系，可以看看答案，只要你能在继续往下学习之前彻底搞懂它。

  • 用你刚学的概念写点自己的程序。这能加强你对知识点的把握，也能加深你的印象。

常见问题解答

Q: 如果我怎么也搞不明白某个概念怎么办？

如果你有的地方不懂，或者卡在某个概念上了，你可以：

  • 看看下面的评论区，可能别人也遇到了同样的问题。（译者注：评论区请到原文网站的对应课程章节查看）

  • 浏览一下接下来的一两个小节或者后面几课，你的问题可能在后面有解答。

  • 查查搜索引擎，看看你的问题或者错误报告有没有在别处有解答。

  • 到某些编程论坛上发帖寻求帮助，比如 Stack Overflow 。

要是还行不通，先放下搞不懂的地方，以后再来看。可能学的更多了之后就会发现以前的问题变简单了。

Q: 以前学过的概念忘了，找不到对应课程在哪怎么办？

请善用网站索引。查找你想看的概念或主题的关键词，然后看看提到它的那些些课程。（译者注：原文网站的索引只接受英文关键词）

还有最后一点小请求：原文网站 LearnCpp 是靠广告支撑的。如果你觉得这个教程还不错，请在原文网站关掉广告屏蔽器。（译者注：如果你英语阅读能力还可以，强烈建议到原文网站去看英文原文，也让原网站作者多一些收入。另外，译文网站这里目前没有任何广告）

好了，让我们开始吧！
